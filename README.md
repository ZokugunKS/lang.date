Extends the built-in Date class.

Getting Started
---------------

### In Node.js

With [node](http://nodejs.org) previously installed:

	npm install @zokugun/lang.date

Use it with `kaoscript`:

```kaoscript
import '@zokugun/lang.date'

const d = Date.tomorrow()
```
